Installing Tensorflow

Pathの../..googleapis.com/tensorflow/..以下は各自のパソコンと何で動かすか(CPU or GPU)を選択します。
それ以降のTensorflowのVersionは、代謝が激しいので最新版(Nightly Build)をインストールしましょう。
最新版出ないと動かない場合が多いです。

you have to install the nightly build
